  • Java开发工程师0

    职位职责

    1.与产品经理、设计师和后端工程师紧密沟通，理解产品设计意图并准确实现需求；

    2.参与锤子科技官网、欢喜云、论坛等网站的开发以及部分 Hybrid 应用开发；

    3.开发内部运营支撑系统，支持公司运营工作，提升运营效率；

    4.配合市场营销，进行相关专题或者活动的开发；

    5.开发内部工具与 UI 组件，改进工作流程，提升团队工作效率。

    我们对您的期望

    1.重视细节，避免自己开发产品的错别字与像素偏差等细节错误；

    2.深刻理解 Web 标准，关注前端技术发展动向；

    3.HTML、CSS、JavaScript 基础扎实，熟悉各种前端技术；

    4.3 年以上大型 Web 应用前端架构与开发经验；

    5.较强的逻辑思维和沟通能力，遇到问题能够快速定位并提供合理解决方案；

    6.认真负责，积极主动，有良好的团队合作意识。

    必备条件

    1.计算机或设计等相关专业背景；

    2.热爱开源技术，参与过开源项目；

    3.有 Hybrid 项目开发经验，擅长移动端开发；

    4.对 Angular、React、Vue 或者 Node.js 等技术有较深入研究。
